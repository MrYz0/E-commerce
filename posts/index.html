# 电商简介🌞

电子商务专业于2005年获批全日制本科招生资格，2006年成为学校第一批重点建设专业。 借助上海第二工业大学工科见长的优势，秉承技术引领商务的理念，我校电子商务专业2015年由原授予“管理学”学位，改授“工学”学位，是上海市目前唯一授予“工学”学位，培养技术与商务相结合的复合型、应用型、创新型人才的全日制本科电子商务专业。 专业多年来注重教学模式创新，陆续获得上海高校全英语示范性课程、上海市优质在线课程、上海市精品课程、上海高校课程思政项目、上海高校优质混合式在线课程示范案例评选示范案例、国家级一流本科线上线下混合式课程荣誉称号；2017开始探索产教融合教育教学模式，校企合作改革获得上海高校本科重点教改项目立项；2019年立项第一批上海高校大学计算机课程教学改革项目，并于2021年获批学校精品领航思政专业。 ​专业成果于2019和2021两次获得上海第二工业大学校长奖，并于2021年获得上海第二工业大学教学成果特等奖。

# 专业教师简介

电子商务专业教师共9名，其中教授1名，副教授3名，实验室教师1名。具有博士学位教师6名，双师型教师6名，海外背景5名。 教师获得荣誉有：全国首批万名优秀创新创业导师、上海市优秀共产党员、上海市育才奖。 教师团队专业领域涵盖了电子商务专业所涉及的技术开发和服务、网络商务运营、商务数据分析等领域，已经形成了理论和实践教学团队、课外创新及竞赛指导团队，学科研究团队等来支撑了电子商务专业建设。 其中省部级及以上立项项目数据如下：教学建设与教改项目18个（国家级6个），出版教材专著6本，科研项目4个（国家级1个），包括1门国家级一流课程、3项教育部产学研合作项目、1项国家社科基金和1项教育部人文社会科学研究项目。

# 一专业定位

电子商务专业要培养具备电子商务应用开发能力、电子商务数据分析能力和电子商务运营能力的应用型的创新技术人才。

# 二培养目标

# 1．培养总目标

​ 本专业培养德、智、体、美、劳全面发展，具备现代管理和信息经济理念、掌握信息技术和电子服务综合技能、具有扎实的专业基础和良好的知识结构，具备一定的互联网创新创业素质，能从事网络环境下商务运营、专业管理和技术服务的复合型、应用型、创新型专业人才。

# 2．价值引领目标

​ 本专业以准电子商务人的职业素养和劳模精神为引领，培养具有良好科学素养、诚信品质和社会责任感，具有善良的人性和科学的理性，具有国际视野和文化融通能力，具有政治意识、大局意识、核心意识、看齐意识、德才兼备的有用之人。

# 三基本要求

# 1．思想政治和德育要求

​ a. 具备良好的道德素养、人文素养、科学素养和职业素养。

​ b. 具备较高的网络文明素养、电子商务诚信与信用素养、信息安全与保密素养。

​ c. 具备良好的人际沟通素质和团队合作精神。

# 2．知识要求

​ a. 牢固掌握现代管理、网络经济和信息技术相关的基本理论和专业知识。

​ b. 掌握电子商务类专业理论与专门方法，能够理论与实践紧密结合，解决实际问题。

​ c. 了解快速发展的电子商务新兴产业动态,注重产业技术创新和商业模式创新并及时掌握相应的理论与技术知识。

# 3．能力要求

​ a. 具备独立自主地获取本专业相关知识的学习能力。

​ b. 具备将所获取的知识与实践融会贯通并灵活应用于电子商务实务的技能。

​ c. 初步具备利用创造性思维方法开展科学研究的能力，以及基于多学科知识融合的创意、创新和创业能力。

​ d. 具备良好的外语听、说、读、写能力。

# 4．职业资格证书类型和名称

​ 阿里巴巴（中国）教育科技有限公司、国家服务外包人力资源研究中心：中国跨境电子商务师三级

# 5．服务面向

​ 面向电子商务企业和相关企业的电子商务部门从事相关技术和管理工作，包括电商网站策划或编辑、电商网站建设和维护、企业商品和服务的营销策划、跨境电商运营、商务智能决策、网站优化、电子商务项目规划和运作等方面工作的应用型专业技术和管理人才。

​ 实行弹性修业年限，一般为四年，弹性幅度最短不低于三年，最长不多于六年。

​ 本专业学生必须按指导性培养计划的要求修读完成各类别课程规定的最低学分、并完成第二课堂规定的所有内容，总学分达到152学分，方可毕业；达到学士学位授予条件者，授予工学学士学位。

# 七，主干学科

​ 计算机科学与技术、管理学、经济学。

# 八，专业核心课程

# 1.电子商务概论（全英文）

电子商务概论是电子商务专业的一门重要专业基础课，主要内容包括：电子商务概念、电子商务发展历程、电子商务类型、电子商务技术、电子商务安全、电子商务支付、电子商务营销、电子商务物流、电子商务法律、电子商务发展趋势、移动电子商务、社交电子商务。通过本课程的学习，学生除了能掌握现有成熟电子商务模式中的各操作环节的具体操作，还能抓住电子商务的本质：信息技术与互联网在商务领域的应用。电子商务的发展不仅仅受到技术发展的影响，法律法规等外部因素对电子商务的发展也有重要的制约作用。课程中融入“一带一路”和“进口博览会”等相关政策及意义，让学生理解中国扩大开放的决心，是应对全球贸易新格局、促进产业结构升级、以信息化带动工业化，以工业化促进信息化，走新型工业化道路的客观要求和必然选择。

# 2.程序设计基础（Java）

​ 程序设计思想、概念、技术、良好的程序设计风格以及过程化程序设计，包括数据类型、控制结构、数组以及各种常用的算法；旨在使学生通过学习，并经过一定的训练和实践，能够掌握程序设计的方法，并具备良好的程序设计风格。课程采用理论和实践相结合、课内和课外相结合的教学方法。课堂教学强调的是思想与方法，实践教学旨在提高学生程序设计及调试的能力，教学手段上融合人文精神和科学精神，教学内容上植入儒家经典字句，在教学中对学生进行计算思维培养中的方法、内容和理念。

# 3.管理学

​ 管理学是经管学院的一门公共基础平台课。通过学习本课程的学习，使学生掌握管理的基本职能、基本原理和基本方法，掌握一般管理科学的基本原理，为今后进一步学习打下一个良好的基础。在学习过程中要求学生了解管理思想发展；计划工作的基本程序与原理；熟练掌握有效决策的过程与方法；掌握组织的基本结构和组织设计的基本原理；人力资源管理的基本框架；激励的基本原理与沟通的基本技巧；控制的基本原理和基本方法；领导工作等。通过该课程学习，促使学生了解和思考社会主义经济环境下企业管理的特性与当前形势下存在问题及解决方法。

# 4.应用统计学

​ 本课程主要研究统计学的基本理论和方法。论述统计调查、统计整理和统计分析等统计工作过程和方法，为学生专业课程的学习打下一定的基础。要求学生通过学习统计学的基本原理和方法，掌握统计工作中调查、整理和分析等环节的主要内容。基本要求是：了解统计学的研究对象，并理解其基本概念；掌握统计调查的基本方法；掌握统计分组与整理；较熟练地掌握总量指标、相对指标和平均指标的概念和计算，熟悉这些综合指标的应用；掌握动态数列的主要分析指标；掌握综合指数的编制方法及因素分析；理解抽样推断以及相关和回归分析方法。通过该课程学习，学生应能够理解当前经济形势下的经济统计问题，并能应用所学知识分析和理解国家的经济政策，认清国际经济形势。

# 5.数据库应用基础

​ 本课程主要讲授数据库系统的基本概念和实际应用，主要内容包括：数据库基础知识、数据库管理系统的安装和配置、SQL语言、数据库和表的操作、事务管理、存储过程管理、视图管理、数据库的高级操作、数据库设计、数据库技术发展趋势等。引导学生要深入学习，将课堂上学到的知识应用于实际，尤其是在数据库的设计时要多考虑客户需求，设计出合理的数据库，引导学生做好数据安全措施和保障，保护用户的隐私信息，树立诚信的优良品质。

# 6.网络营销

​ 通过本课程的学习，使学生掌握电子商务营销的基本理论与方法，学会使用网上营销的服务工具并设计站点，使用各种电子商务营销的定价策略，建设电子商务营销渠道，网络促销方法和运用网络广告等基本技术，了解电子商务营销的实施、组织与控制。教学中融入诚信经营、抵制假货等经营理念，同时从促销方法讲解和网络广告制作宣传中培养学生正能量传递的习惯，不发布虚假信息，用产品质量来保障产品品牌的理念。

# 7.电子商务系统分析与设计方法

​ 本课程目前采用在线结合课堂教学的模式，是电子商务专业一门重要的专业课程，涉及面广，主要讨论电子商务系统的分析与设计过程的原理、方法、技术、工具和应用。课程内容包括电子商务系统构建时所涉及的相关知识，系统分析和设计的基本概念和原理，系统开发生命周期过程，重点教授学生系统分析和系统设计阶段所需要掌握的知识点，包括调研方法、数据流图、实体关系、功能流程、模块设计等等，通过团队练习来巩固理论知识，同时培养学生的团队协作能力。在教学中对学生说明学习新技术的重要性和从事电子商务人员道德品质的重要性，引导学生培养诚信、团结的意识。

# 8.电子商务物流

​ 电子商务物流是一门专业选修课，主要内容包括电子商务物流概述、电子商务物流市场及物流模式、电子商务物流配送管理、电子商务物流技术基础、电子商务物流信息管理系统、供应链管理、物流管理方法、物流成本核算等内容。通过本课程学习，使学生能了解电子商务物流运作与管理的基础知识，同时通过对一些案例的分析力求扩展学生的思维空间，加深学生对理论知识的理解，同时增强学生自主学习的能力。通过该课程学习，学生应能够增强专业素养，理解国家的“一带一路”政策的意义，理解绿色物流、循环经济的概念，并在生活工作中践行。

# 9.商务智能

​ 本课程从商务智能的基本概念和应用开始，要求学生熟练了解商务智能的核心技术，包括理解商务智能系统架构、数据仓库、OLAP和数据挖掘的基本技术内容。在此基础之上了解商务智能在移动商务、知识管理、Web挖掘、绩效管理和电子商务等领域的应用。理解商务智能在未来发展的趋势。理解商务智能在未来发展的趋势，在课程讲解中引用近年来中国快速发展的各行各业数据，在进行数据挖掘，商务决策的同时感受祖国的快速发展和强大，有利于培养同学的爱国主义精神。

# 10.电子商务安全技术与管理

本课程主要讲授了电子商务安全的含义、电子商务系统的各种安全隐患及其解决方案。主要内容：电子商务安全的原则、目标、策略及实现、商务平台常见的攻击以及常用的防御方法、公钥密码体制及算法、数字签名及公钥证书、电子支付安全及安全协议，交易信息安全及其解决方式，电子商务安全技术的发展趋势。教育学生要遵守安全方面的法律法规，用技术和管理防止对电子商务交易和国家财产安全的破坏，融入爱国主义教育和道德教育，使学生成为德才兼备、有职业道德的应用型人才。

# 九，主要实践环节

​ 电子商务专业导论实践、视觉设计实践、网店美工设计实践、商务静态网站设计实践、电商网店操作实践、商务动态网站开发实践、数据统计分析工具实践、新媒体运营实践、搜索引擎营销与优化实践、电子商务综合项目开发实践、跨境电商平台运营综合实践、电子商务创新创业实践、互联网开发技术实践、商务智能与数据挖掘项目实践、电子商务企业顶岗实践、电子商务创新创业实践、电子商务企业顶岗实践和专业职业技能实践。

# ★1，专业模块课和实践模块课的选修说明：

专业课按照不同的能力要求分模块设置，学生必须选修其中1个模块并达到该模块要求的学分。专业实践模块必须按对应的专业课模块选修。

# 1，模块A：移动开发模块

A模块方向注重深入学习H5技术、Android移动开发及SpringMVC框架技术和数据分析等知识，具备UI设计、移动APP应用设计、商务网站开发和维护、移动互联产品设计、测试和软件项目管理等能力，培养熟悉电子商务流程，能够进行相关程序开发岗位和软件技术支持的综合性技术人才。

# 2，模块B：跨境电商模块

B模块方向注重深入学习跨境电商实务与操作、跨境电商跨境电商物流选择方案以及动态规划、跨境电商平台数据分析和海关通关等知识，具备为企业开展跨境电商业务提供解决思路、明确开展跨境电商的条件、掌握目标市场的需求与营销策略等能力，培养跨境电子商务（出口）平台外贸岗位、跨境电子商务（出口）平台外贸运营与推广岗位等跨境电子商务亟需人才。

# 3，模块C：数据分析模块

C模块方向注重深入学习商务数据模型设计与决策分析、电商用户行为数据的分析和定向营销、电商数据的可视化设计和电商大数据的分析应用等知识，旨在培养电子商务运维数据分析师、电子商务网站数据分析师、数据软件技术支持技术员等专业人才。

# ★2，职业资格证书与课程的关联说明：

学生通过网络营销、电子商务物流、互联网金融与支付、国际贸易理论与实务、跨境电子商务理论与实务（双语）、电商网店操作实践、新媒体运营实践、跨境电商平台运营综合实践课程的学习，可参加与本专业相关的职业资格证书考核：全国计算机信息高新技术考试（跨境电商模块）（中级）、中国跨境电子商务师三级资格证书。

学生取得全国计算机信息高新技术考试（跨境电商模块）（中级）资格证书、中国跨境电子商务师三级资格证书，可申请跨境电子商务理论与实务（双语）、跨境电商平台运营综合实践、专业职业技能实践课程免修并获得相应学分。
